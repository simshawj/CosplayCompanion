# Cosplay Companion Android App
Android app to go with the Cosplay Companion website.

Cosplay Companion currently is a Ruby on Rails API backend and Android App.

Compatible Android SDKs: 15-24

External libraries:
* Picasso
* Retrofit
* Butterknife
* Dagger2

Testing Libraries:
* Robolectric
* Mockito
* Hamcrest
